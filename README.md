# STARCOM SOP Swimlane V5

Versi ini menghapus panel pilihan/notifikasi di samping. Setiap decision gate langsung menampilkan dua jalur pada bagan:

- GO / LOLOS: menuju proses berikutnya.
- NO-GO: menampilkan status, solusi/tindakan koreksi, dan tujuan eskalasi.

Jalankan dengan:

```bash
streamlit run app.py
```
