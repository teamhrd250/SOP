# Starcom SOP Sales to After Sales — Standalone

Versi ini tidak membaca Excel atau file data eksternal. Cukup upload seluruh isi folder ke root repository GitHub dan deploy `app.py` di Streamlit Community Cloud.

Struktur minimum:

```
app.py
requirements.txt
.streamlit/config.toml
```

Setelah commit, pada Streamlit Cloud pilih **Reboot app**. Jika aplikasi lama masih tampil, lakukan **Clear cache** lalu **Reboot**.
